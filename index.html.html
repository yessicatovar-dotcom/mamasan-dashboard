Nota sobre IA externa

La versión anterior intentaba llamar a Claude desde el navegador con una API key visible. Eso suele fallar por seguridad/CORS y además expone la clave. Para usar Claude real, conecta este dashboard a un backend o función serverless que guarde la key en servidor y devuelva solo el análisis.
